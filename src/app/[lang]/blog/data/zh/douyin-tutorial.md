# 抖音下载视频教程

> 基于 web 操作流程：
>
> 1.从社交媒体中获取 url 链接
>
> 2.将社交媒体的 url 链接粘贴的到网站输入框中
>
> 3.下载图片或视频到本地

## 1.如何从抖音中获取媒体 url 链接？

- 在您想要下载的视频右侧找到分享按钮，点击复制链接

## 2.将链接粘贴到[SnapVee](https://snapvee.com/)[ ](http://snapvee.com/blog/tutorial)网站输入框中

## 3.下载图片或视频到本地

- 点击下载，等待视频解析成功，在弹出的窗口中点击下载视频，将视频保存到本地即可
